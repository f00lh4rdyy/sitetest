**Запуск проекта**
- Скопировать себе копию проекта:
    ```sh
    $ git clone https://github.com/f00lh4rdyy/sitetest.git
    ```
- Создать виртуальное окружение и активировать его:
    ```sh
    $ virtualenv --python=python3 venv
    ```
    ```sh
    $ source venv/bin/activate
    ```
- Запустить проект:
    ```sh
    $ flask run
    ```
